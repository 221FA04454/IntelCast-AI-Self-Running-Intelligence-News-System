# D-ID AI Presenter Setup Guide

## What is D-ID?
D-ID is the industry-leading platform for creating realistic AI talking head videos with perfect lip-sync. It's used by major news organizations and content creators.

## Setup Instructions

### Step 1: Create Account
1. Go to: https://studio.d-id.com/
2. Click "Sign Up" (free trial available)
3. Verify your email

### Step 2: Get API Key
1. Log in to D-ID Studio
2. Click your profile (top right)
3. Select "Settings"
4. Navigate to "API" tab
5. Click "Create API Key"
6. Copy your API key

### Step 3: Configure
1. Open: `d:/interest/ai_presenter_agent.py`
2. Replace `D_ID_API_KEY = "YOUR_D_ID_API_KEY_HERE"`
3. Paste your actual API key
4. Save the file

### Step 4: Test
```bash
cd d:/interest
python ai_presenter_agent.py
```
